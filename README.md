#  🧮 Calculator 만들기 🧮

> ### 프로젝트 소개
- 기본적인 사칙연산(+,-,*,/)을 수행하는 계산기 구현
- 사용자로부터 숫자와 연산자를 입력받아 결과 출력
- 연산 결과는 리스트에 저장하여 누적 관리

> ### STEP1. 클래스 없이 기본적인 연산 수행 계산기 제작
 
### 주요 기능
0) 기본 클래스 내 main 메서드에서만 구현 
1) 첫번째 숫자 입력 (0이상의 정수가 아니면 재입력 요청)
2) 두번째 숫자 입력 (0이상의 정수가 아니면 재입력 요청)
3) 사칙연산 입력 (+, -, *, / 가 아니면 재입력 요청)
4) "exit" 입력을 하면 프로그램 종료, 그렇지 않으면 1~3 반복 수행


> ### STEP2. 클래스 적용해 기본적인 연산 수행

### 주요기능
1. 전체적인 기능 및 흐름은 STEP1 과 동일
2. 사칙연산을 별도의 Calculator라는 Class를 생성하여 구현
3. 사칙연산 결과값을 resultList 컬렉션(ArrayList)에 할당하여 표시
4. 결과 리스트에 강제로 [1,2,3,4,5] 데이터를 입력하는 기능 구현
5. 가장 먼저 저장된 데이터 삭제 기능 구현
